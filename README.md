# 🚀 30 Days of JavaScript Challenge

Welcome to my **30 Days of JavaScript Challenge** repository!
This project represents a complete journey through the fundamentals of JavaScript, built through consistent daily practice and hands-on mini-projects.

---

## 📌 Overview

Over the course of 30 days, I focused on strengthening my JavaScript skills by working on practical exercises and small projects. Each day introduces new concepts, progressively building toward more advanced topics.

This repository serves as:

* A **learning journal**
* A **reference for core JavaScript concepts**
* A **collection of mini-projects**

---

## 🧠 Topics Covered

The challenge includes a wide range of JavaScript concepts:

* Variables & Data Types
* Operators & Expressions
* Conditional Statements (`if`, `switch`)
* Loops (`for`, `while`, `do...while`)
* Functions (declaration, expression, arrow functions)
* Arrays & Array Methods
* Objects & Object Methods
* DOM Manipulation
* Events Handling
* ES6+ Features (let/const, template literals, destructuring, spread/rest)
* Error Handling
* Asynchronous JavaScript (Promises, `async/await`)
* Basic Algorithmic Thinking
